# 🌐 React Portfolio Website

This is my **personal portfolio website** built using **React.js** and **Tailwind CSS**.
It showcases my skills, projects, and contact information as a **Frontend Developer**.

---

## 🚀 Live Demo

🔗 https://portfolio-tailwindcss-use.vercel.app/

---

## 👩‍💻 About the Project

This portfolio website is designed to present my work and skills in a modern and responsive layout.
It includes sections like **Home, About, Projects, and Contact** to highlight my experience and projects.

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Tailwind CSS

* 🎯 CSS3
* ⚡ Vite

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Aditiv2004/Portfolio.Tailwindcss_use.git
```

Install dependencies:

```
npm install
```

Run the project:

```
npm run dev
```
